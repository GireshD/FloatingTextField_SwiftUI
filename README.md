# FloatingTextField_SwiftUI

Floating TextField In SwiftUI

Use 
FloatingTextFieldView to show floating textfield 


Ex:

@State var email: String = ""
FloatingTextFieldView(leftIcon:"person",placeHolder: "Enter Email", text: $email)
